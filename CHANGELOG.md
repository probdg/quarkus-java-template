# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial Quarkus Java Template with best practices
- Quarkus 3.17.5 with Java 17
- RESTful API with example endpoints
- Comprehensive testing setup (JUnit 5, REST Assured)
- Code quality tools (Checkstyle, PMD, SpotBugs)
- JaCoCo code coverage
- SonarQube integration
- GitHub Actions CI/CD pipeline
- Docker and Docker Compose configuration
- PostgreSQL database integration with Panache ORM
- Health checks (liveness and readiness)
- Prometheus metrics
- OpenAPI/Swagger documentation
- Structured logging (JSON support)
- Input validation with Bean Validation
- Exception handling
- Comprehensive README with quick start guide
- Contributing guidelines
- Security policy
- Apache 2.0 License
- EditorConfig for consistent coding style
- Git pre-commit hooks setup script
- Environment variable configuration

## [1.0.0] - 2026-02-13

### Added
- Initial release of Quarkus Java Template
- Production-ready structure following industry best practices
- Inspired by [nuxt-template](https://github.com/sadigitID/nuxt-template)
